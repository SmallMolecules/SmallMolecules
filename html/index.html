Particle Simulator
A 3D particle system visualiser for educational purposes
About this Framework

The scope of this framework is to create a working particle simulation in C#, to be implemented in Unity. This page specifies each classes and structs within the framework - the public and private attributes and methods.

The overall approach was to encapsulate iscolated instnaces of simulators within a simulator manager. Each simulator contains its own independent set of Particles, Scales and Static and Dynamic fields that all work together to produce a simulation. To see the list of classes, click here. To see the directory structure click here.

Installation

To install this framework, first you need to install unity on your system. To do that, download version 2020.3.16 from Unity's Downloads Page. Alternatively if you already have a working install of Unity, you can download and install version 2020.3.16 from Unity Hub.
