Detailed Description

Struct for holding scale data.

This Struct is used effectively as a datatype, where the scale value (VAL) is stored separately as an coefficient (COEFF) and an exponent (EXP). For example 0.0001223 is represented in a Scale as COEFF = 1.223, EXP = -4 and VAL = 0.0001223
